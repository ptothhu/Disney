  19. What is the difference between Page Views, Page Visits and Unique Visitors of a web page? How would you count them in order to get a reliable result?

    20. Page views represent the total number of pages that visitors looked at on our site.
    Page visits represent the number of times the website was visited, without regard to repeat visitors.
    Unique visitors represent the number of actual people that visited our site.
    If you visit a website today and look at 3 pages, that would count as 1 visit and 3 page views.
    If you return to the same website another day and look at the same pages again, that will count as 1 more visit and 3 more page views. However, it will only count as 1 visitor since it is the same person visiting twice.
    If someone else visits the same website today and looks at those pages that will count as 1 visit, 1 visitor, and 3 page views.

  20. What is the difference between a HTTP 301 and a 302 redirect?

    21. Both are HTTP redirectional status codes indicating that the client must take additional action to complete the request.
    HTTP 301: Permanent URL redirection. Current links or records using the URL that the response is received for should be updated.
    HTTP 302: Found but temporarily redirected. The browser is invited by a response with this code to make an identical request to the new URL specified in the location field.
